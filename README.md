# Secure Password Manager

## Overview

The Secure Password Manager is a desktop application designed to safely store and manage user credentials such as usernames, passwords, and notes. Access to the system is protected using a Master Password, ensuring that only authorized users can view stored data.

All sensitive information is encrypted before being saved, so no passwords are stored in plain text at any stage.

## Features

- Master Password authentication
- Secure login system
- Encrypted storage of credentials
- Store usernames, passwords, and notes
- View, add, and manage saved entries
- Protection against unauthorized access
- Local secure database storage

## Security Features

- Data encryption before storage
- Passwords are never stored in plain text
- Master password required for access
- Secure credential handling

## Technologies Used

- Python
- Tkinter (GUI)
- Cryptography / Encryption Library
- File-based / Database storage (depending on your project)

## Purpose

This project was developed as an Information Security (IS) academic project to demonstrate secure data storage, encryption techniques, and authentication mechanisms in desktop applications.
